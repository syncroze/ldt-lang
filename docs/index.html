ldt-lang

Logic Driven Text Language — a small, dependency-free embeddable templating engine for PHP. Everything is literal text until a bracket construct appears.

GitHub README Caveats Design history File structure

Assign & interpolate

[set greeting]Hello[/set][set name = world]@{greeting}, @{name}!

[set fruit. = apple][set fruit. = banana][set fruit. = cherry]
Second fruit: @{fruit.1}

Conditionals

[set role = editor]
Role: [if @role == admin]full access[elseif @role == editor]can edit[else]read only[/if]

[set email = ada@example.com]
Email ok : [if @email contains "@"]yes[/if]

Loops

[set fruit. = apple][set fruit. = banana][set fruit. = cherry]
[for f in @fruit]
  - @{f}
[/for]

Evens: [for n in 2 to 10 by 2]@{n} [/for]

Expressions

[set price = 20][set qty = 3]
Subtotal: @(@price * @qty)

[set tags. = a][set tags. = b][set tags. = c]
Tag count: @(count @tags)

Filters

[set name]  ada lovelace  [/set]
trim + upper : @{name | trim | upper}

who    : @{missing | default: "guest"}

Comments & escaping

[# a comment — renders to nothing #]

[set x = hi]
literal \@{...} : \@{x}          [# renders @{x}, not interpolated #]
literal tag     : \[set a = 1]   [# printed as text, not executed #]
